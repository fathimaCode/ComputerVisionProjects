About Us

This web application is built using the Flask framework. It enables users to upload video files, which are then analyzed by a pre-trained deep learning model for fire detection. The front end of the application uses HTML templates rendered via Flask's templating engine, with routes to handle various functions like video upload, prediction display, and login.
